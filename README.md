# Wedding Management Frontend

Frontend application for the Wedding Management System built with React + Vite.

---

## Tech Stack

- React 18
- Vite
- TailwindCSS
- Material UI (MUI)
- Radix UI
- React Router
- React Hook Form
- Recharts

---

## Backend Repository

Backend API:

```txt
https://github.com/your-username/wedding-management-backend
```

Example API URL:

```env
VITE_API_URL=http://localhost:5000/api
```

---

## Installation

Clone project:

```bash
git clone https://github.com/your-username/wedding-management-frontend.git
```

Move to project folder:

```bash
cd wedding-management-frontend
```

Install dependencies:

```bash
npm install
```

or

```bash
pnpm install
```

---

## Run Development Server

```bash
npm run dev
```

or

```bash
pnpm dev
```

Frontend runs at:

```txt
http://localhost:5173
```

---

## Build Production

```bash
npm run build
```
